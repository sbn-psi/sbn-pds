OSIRIS-REx OTES Bundle

This bundle collects all the operational data products produced by the OSIRIS-REx Thermal Emission Spectrometer (OTES). OTES is used for the spectral characterization of the surface of (101955) Bennu.

The bundle now includes all data through Nov. 17, 2020.

Release dates:

  • Initial release June 28, 2019: Data through Nov. 17, 2018
  • Release on Dec. 12, 2019: Data through May 17, 2019
  • Release on Feb. 25, 2020: Data through Aug. 17, 2019
  • Release on May 27, 2020: Data through Nov. 17, 2019
  • Release on Aug. 25, 2020: Data through Feb. 17, 2020
  • Release on Dec. 1, 2020: Data through May 17, 2020
  • Release on Mar. 5, 2021: Data through Aug. 17, 2020
  • Release on June 1, 2021: Data through Nov. 17, 2020
