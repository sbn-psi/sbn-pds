OSIRIS-REx TAGCAMS Bundle

This bundle collects all the operational data products produced by the OSIRIS-REx Touch-and_Go Camera Suite (TAGCAMS). TAGCAMS is a suite of engineering cameras used for optical navigation (navcam), natural feature tracking (nftcam), and documenting sample stowage (stowcam).

The bundle now includes all data through Aug. 17, 2019, covering most of the way through Orbit B.

Release dates:

  • Initial release on Feb. 15, 2019: Outbound Cruise (including Earth/Moon)
  • Release on July 1, 2019: Bennu data through Nov. 17, 2018
  • Release on Aug. 27, 2019: Bennu data through Feb. 17, 2019
  • Release on Nov. 25, 2019: Bennu data through May 17, 2019
  • Release on Feb. 25, 2020: Bennu data through Aug. 17, 2019

Note: Select "Smaller downloads" to choose zip files for individual processing level and mission phase, or a zip file tagcams_metadata.zip containing all documents and metadata.
