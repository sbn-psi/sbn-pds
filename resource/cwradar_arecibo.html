Arecibo Radar Doppler Spectra of Asteroids

This data set contains radar Doppler spectra taken with the Arecibo Observatory Planetary Radar saved as a set of comma-separated-value tables each in a singe file. This version contains the data described in Virkki et al. (2022).

Release Date: 2024-12-18
