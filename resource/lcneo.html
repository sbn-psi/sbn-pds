Ondrejov Near Earth Asteroid Lightcurves

This data set contains photometric lightcurves of near-earth asteroids obtained by Petr Pravec and his collaborators at Ondrejov Observatory from 1984 through 1998. Lightcurves for 42 asteroids are included.

Release Date: 2020-02-28

Data Set Info
PDS4 version urn:nasa:pds:gbo.ast-neo.ondrejov.lightcurves::1.0 Browse Download (826 kb)
PDS3 version EAR-A-3-RDR-NEO-LIGHTCURVES-V1.1 Browse Download (648 kb)

The PDS4 bundle was migrated from the PDS3 data set in February 2020. The data content is unchanged.

Use the following citation to reference this data set:

"Pravec, P., Sarounova, L., Wolf, M., Harris, A.W., Mottola, S., Hahn, G., Erikson, A., and Young, J.W. (2020). Near Earth Asteroid Lightcurves V1.0. urn:nasa:pds:gbo.ast-neo.ondrejov.lightcurves::1.0. NASA Planetary Data System; https://doi.org/10.26033/f87t-qr41."
