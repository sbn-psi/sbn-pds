Hayabusa AMICA Images with Geometry Backplanes

The Asteroid Multi-band Imaging Camera (AMICA) of the Hayabusa mission to the asteroid 25143 Itokawa obtained 1662 images from May 11, 2003, shortly after launch, until November 19, 2005, after Itokawa encounter. This data set provides derived data products for 1339 of those images that includes geometric information for each pixel.

Data Set Info
HAY-A-AMICA-3-AMICAGEOM-V1.0 Browse (for download, see below)

To facilitate downloading of this large data set, it has been divided into smaller compressed pieces. To reconstruct the data set on your machine, download 'amicageom0.zip' and unzip it to create a copy of the data set without the data directories. Download 'ddr_gaskell.tar' and 'ddr_spice.tar', place them in the data directory of the data set, and un-tar them to populate the data directory. Within the two data subdirectories, the data files themselves are gzipped. If you unzip them all the full size of the data set will be approximately 82 Gbytes.

  • amicageom0.zip (51 Mb)
  • ddr_gaskell.tar (4.4 Gb)
  • ddr_spice.tar (2.4 Gb)

Use the following citation to reference this data set:

Barnouin, O. and Kahn, E., Hayabusa AMICA Images with Geometry Backplanes V1.0. HAY-A-AMICA-3-AMICAGEOM-V1.0. NASA Planetary Data System, 2012.
