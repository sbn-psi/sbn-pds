MSX Zodiacal Dust Data

urn:nasa:pds:msx.zody.dust V1.0

The Midcourse Space Experiment (MSX) mid-infrared emission measurements from the zodiacal dust cloud in spectral bands centered at 8.3 12, 15, and 21 microns.

Instructions for download:

This data set is too large for easy download by most browsers in a single file. Follow these instructions to download and re-assemble the entire data set:

1. Download the four zip files above. The larger ones may take a while.

2. Unzip the file msxzody.zip in the place where you want the data set. It will create its own directory, called msxzody, and all the subdirectories and catalog files of the data set.

3. The "data" directory of the resulting msxzody directory structure will be empty. Place the three remaining zip files in this directory.

4. Unzip each of the three data zip files in the data directory. They will create the appropriate subdirectories for the data.
