Overview

  • Abstract: This bundle contains global and regional topography of Saturnâ€™s moon Tethys, as well as photometric data for each of the regional models. Data products were generated using the Gaskell Stereophotoclinometry (SPC) software suite with images from the Cassini mission to Saturn. The global model is an update to the archived "Gaskell Tethys Shape Model", and is generated with more images and has higher resolution topography in some locations. In addition to the topographic models, we also provide assessment data to understand the uncertainty of the models.
