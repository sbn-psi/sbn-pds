Overview

  • Abstract: This dataset contains the data used by the OSIRIS-REx mission to test and validate the software tools needed to generate shape models of the target asteroid, Bennu. Several synthetic truth models were created as simulated Bennu asteroids for testing. This package contains these truth models as well as the resulting models that were generated by the software during the test.
