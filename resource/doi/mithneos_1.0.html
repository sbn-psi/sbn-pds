Overview

  • Abstract: This dataset is comprised of the near-infrared (0.65-2.5 micron) spectra of near-Earth objects (NEOs) and Mars crossing asteroids (MCs) as part of the MIT-Hawaii Near-Earth Object Spectroscopic Survey (MITHNEOS). The spectra were obtained from the 3-meter NASA Infrared Telescope Facility (IRTF) on Mauna Kea using the SpeX instrument in Low-Resolution Prism mode and the 0.8 x 15" slit. Guiding was performed using spillover light from the slit with GuideDog for data taken prior to 2014, and using the MORIS CCD imager 2014 and later.
