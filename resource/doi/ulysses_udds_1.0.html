Overview

  • Abstract: This bundle contains the data from the Ulysses dust detector system (UDDS) from start of mission through the end of mission, 1990-2007. (As the dust detector was turned off after Nov. 30, 2007, this is the last date for which UDDS data is recorded.) Included are the dust impact data, noise data, laboratory calibration data, and location and orientation of the spacecraft and instrument.
