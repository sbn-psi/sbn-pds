Overview

  • Abstract: The V6.0 version of this data set comprises the 2002 Supplemental IRAS Minor Planet Survey (SIMPS) which presents additional and revised diameters and albedos for asteroids detected by the 1983 IRAS mission. This version supersedes versions 1.0 through 5.0, which represent the original 1992 IRAS Minor Planet Survey (IMPS).
