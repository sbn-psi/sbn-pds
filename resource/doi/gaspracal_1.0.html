Overview

  • Abstract: This dataset includes the Galileo Orbiter Solid State Imaging data on the asteroid 951 Gaspra. The raw data have already been archived in PDS with the data set name 'Galileo Imaging (SSI) Asteroid, Earth and Moon Experiment Data Records' and can be found by searching on the data set identification 'GO-A/E-SSI-2-REDR-V1.0'. Only those images in which Gaspra actually appears have been included here. Images in both FITS and ISIS Cube format are provided.
