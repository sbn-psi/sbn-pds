Overview

  • Abstract: This bundle contains a shape model for the Saturnian moon Mimas, along with quality assessment data. The global model is an update to the archived "Gaskell Mimas Shape Model", and is generated with more images and has higher resolution topography in some locations. The data is provided in multiple formats.
