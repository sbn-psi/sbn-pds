Binary Minor Planets Compilation

The data set lists orbital and physical properties for well-observed or suspected binary/multiple minor planets including the Pluto system, compiled from the published literature as inspired by Richardson and Walsh (2006) and similar reviews (Merline et al., 2003; Noll, 2006; Pravec et al., 2006; Pravec and Harris, 2007; Descamps and Marchis, 2008; Noll et al., 2008; Walsh, 2009). In total 317 companions in 300 systems are included. Data are presented in three tables: one for orbital and physical properties; one for companion designations, discovery information, and reference codes for data values; and one giving full references for each reference code. This data set is complete for binary/multiple components reported through 31 March 2017.
