Asteroid Densities

This data set contains a tabulation of asteroid masses, diameters, and bulk densities compiled by D. T. Britt and published in Table 1 of Britt et al. (2002) in the 'Asteroids III' volume. This data set is superseded by the more recent Asteroid Masses data set (EAR-A-COMPIL-5-ASTMASS-V1.0).
