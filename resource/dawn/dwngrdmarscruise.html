Dawn GRaND Mars Flyby

Status - Archived

Raw (Level 1a) Data:

The GRaND EDR are a time-ordered collection of gamma ray and neutron counting data and histograms acquired by GRaND during all phases of the Dawn mission. The dataset includes state-of-health data, such as temperature and voltage readings, needed for the analysis of the counting data.

Dawn GRaND Raw Mars Browse Download (386 Mb)

Use the following citation to reference this data set:

"Prettyman, T.H., DAWN GRAND RAW (EDR) MARS FLYBY COUNTS V1.0. DAWN-M-GRAND-2-EDR-MARS-COUNTS-V1.0. NASA Planetary Data System, 2009."

Calibrated (Level 1b) Data:

The GRaND RDR are a time-ordered collection of corrected gamma ray and neutron counting data and calibrated pulse height spectra acquired by GRaND during all phases of the Dawn mission. This data set is specific to Mars Gravity Assist (MGA), and includes data acquired during Mars approach and flyby. The RDR is a calibrated data set derived from the Experiment Data Records (EDR), consisting of time-series counting rates from which elemental abundances can be determined. Ancillary ephemeris and pointing data needed for analysis and mapping of the time series data are included in the data set.

Dawn GRaND Calibrated Mars Browse Download (17 Mb)

Use the following citation to reference this data set:

"Prettyman, T.H., DAWN GRAND CALIBRATED MARS FLYBY COUNTS V1.0. DAWN-M-GRAND-3-RDR-MARS-COUNTS-V1.0. NASA Planetary Data System, 2009."

Dawn GRaND Cruise

Status - Archived

This data set covers the Dawn GRaND Level 1A data from the Cruise phase. The first volume, DWNXGRD_1A, includes pre-Vesta cruise. The second volume, DWNXCGRD_1A, includes Vesta-Ceres cruise.
