Dawn at Vesta - VIR

The Dawn spacecraft visited the asteroid 4 Vesta for the time period May 3, 2011 to Sept. 5, 2012. VIR is an imaging spectrometer that combines two data channels in one instrument, Visible (VIS) and Infrared (IR). The data sets linked below include raw (level 1a) and calibrated (level 1b) versions of all available images acquired during the Vesta Encounter.

Browse and search the PDS Dawn Vesta archive using the Small Bodies Image Browser.

Calibrated VIR Vesta Version 1.0, superseded:

The Version 1.0 VIR level 1B data has the 3 micron region set to zero (specifically, the range 2.534 - 3.272 microns). The calibration contained artifacts in this range which the VIR team believed made the V1.0 calibrated data unusable. See Version 2.0 above for an updated calibration with the 3 micron region restored.
