Dawn at Ceres - VIR

Dawn VIR is an imaging spectrometer that combines two data channels in one instrument, Visible (VIS) and Infrared (IR). The data sets linked below include raw (level 1a) and calibrated (level 1b) versions of all available VIR data acquired during the Ceres Encounter.

Data Set Info

Browse and search the PDS Dawn Ceres archive using the Small Bodies Image Browser.
