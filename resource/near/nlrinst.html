Highlighted links:

  • NLR Along-Track Profiles - The NAV/NLR team generated along-track profile plots in JPEG format. Each image represents one Level 2 time series of calibrated ranges for a daily profile. Each image contains multiple along-track profile plots including the radius, longitude, and latitude of Eros at bounce points as well as range measurements. The name for each daily JPEG file contains the year and day of year of the observations.
