Radar shape models of asteroids compiled by Lawrence

This data set contains three-dimensional shape models for asteroids observed by ground-based radar facilities. The data set contains shape models for the following asteroids: (2100) Ra-Shalom, (4486) Mithra, (4660) Nereus, (10115) 1992 SK, (29075) 1950 DA (Retrograde and Prograde models), (33342) 1998 WT24, (54509) YORP, (66391) 1999 KW4 Alpha (primary) and Beta (secondary), (136617) 1994 CC Alpha (primary), (276049) 2002 CE26 Alpha (primary), and (341843) 2008 EV5.

Release Date: 2020-03-03
