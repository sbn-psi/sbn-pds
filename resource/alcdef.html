Asteroid Lightcurve Data Exchange Format (ALCDEF) Database

The Asteroid Lightcurve Data Exchange Format (ALCDEF) database contains metadata and data produced by asteroid time-series photometry by amateurs and professionals and submitted to the database using a format that follows ALCDEF standards definition.

Release Date: 2021-09-10
