Small Bodies Data Ferret

Ferret is a web application for searching across multiple small bodies datasets within the Planetary Data System. We've done all the processing of PDS3 and PDS4 datasets, and extracted values from archived tables, so you can access that data directly.

Primary features:

  • Search all data by target name
  • Find and filter small bodies by their taxonomy, family, or physical properties
  • Perform advanced queries using combined attributes from all datasets (Beta - in development)
Open

Ferret is actively being maintained and developed. Send us feature requests or report an issue at ferret-support@psi.edu
