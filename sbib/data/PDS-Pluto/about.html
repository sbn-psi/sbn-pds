About SBIB - Pluto

Pluto Mission Phase covered in SBIB

  • Encounter - Flyby; varying resolution; LORRI: 2015-07-07 to 2015-07-14; MVIC: 2015-07-13 & 2015-07-14

Map Projection and Arc-GIS

Data can be imported into ArcMAP 10.1 by downloading the ISIS Map cubes.
Images (PNG and thumbnails) were generated with an equirectangular projection.

Maps

The overview map is a decreased resolution image of this Pluto Mosaic.

Data processing

We used the USGS software tool Integrated Software for Imagers and Spectrometers (ISIS) as the main tool to process the data. We used a sphere with a radius of 1188.3 km for the shape and the ISIS software routines to convert images into map-based projections.

Coordinates

This software uses the coordinate system as defined by the IAU and used by New Horizons with the prime meridian as the longitude facing Charon (which puts the heart at about 180deg).
